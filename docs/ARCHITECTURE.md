# CLASSGUARD — Architecture

> Validate classification banner markings (CUI/CONFIDENTIAL/SECRET) in documents per portion-marking rules.

```
input ──▶ collect ──▶ rules/analyzers ──▶ score ──▶ findings ──▶ table · json
                              │                          │
                         (this repo)                 MCP tool (agents)
```

- **collect** normalizes the target (file/dir/API) into records.
- **rules/analyzers** apply the heuristics shipped in `classguard/core.py`.
- **score** ranks by severity.
- **MCP server** (`classguard mcp`) exposes `scan` for Cognis.Studio agents.

Extend by adding a rule + a test + a `demos/NN-*/SCENARIO.md`. See [CONTRIBUTING.md](../CONTRIBUTING.md).
